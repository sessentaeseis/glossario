Tags HTML

As tags HTML são os blocos fundamentais da construção de páginas na web. Elas servem como marcadores que indicam ao navegador como os conteúdos de um site são organizados, exibidos, interpretados etc. Cada tag possui uma função específica, desde estruturar o documento até inserir imagens. Elas são escritas dentro de colchetes angulares (< >), e na maior parte do tempo vêm em pares (embora algumas só necessitem de um par de colchetes, como você verá a seguir): uma de abertura (<>) e outra de fechamento (</>). Abaixo, as tags mais fundamentais estão listadas e separadas de acordo com o propósito delas.

Estrutura

Elas determinam como a página é organizada internamente, separando informações de configuração (no cabeçalho) do conteúdo visível (no corpo). Com essas tags, é possível indicar o início e o fim do documento, definir metadados importantes e estruturar o espaço onde o conteúdo será exibido para o usuário.

  • <!DOCTYPE html> - Declara o tipo de documento (HTML5), é a primeira linha do arquivo html. Vale notar que editores como o VSCode tem atalhos (no caso do VSCode, ! e enter) para automaticamente criar o esqueleto do documento HTML.
  • <html></html> - Raíz de todo o documento HTML, contém também a linguagem da página. Assim como o !doctype, pode ser automaticamente criado por um atalho como no VSCode com ! e enter.
  • <head></head> - Contém metadados, links e título. Aqui, o arquivo css é linkado (com <link>) e alguns dados são precarregados, como fontes não-nativas.
  • <link> - Conecta o arquivo a recursos externos, como outros sites. Inclui rel="", que define o quê aquele link faz (ou com quem se relaciona na página). Alguns dos mais utilizados incluem rel="stylesheet" (para linkar um arquivo .css) e rel="shortcut icon" (para definir um favicon).
  • <body></body> - Tudo o que é visível na página está dentro do body. Como o nome diz, é o corpo do texto, responsável por conter o conteúdo do site.
  • <title></title> - Define o título da página na aba do navegador.

Exemplo:

<!DOCTYPE html> <!--Declara o tipo de documento.--> 
<html lang="pt-br"> <!--Declara a linguagem da página.-->  
<head></head> <!--Abertura da tag head, onde informações como metadados são inclusas.--> 
    <meta charset="UTF-8"> <!--Define os caracteres suportados pelo arquivo. UTF-8 é o padrão.-->  
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!--Define a escala do site.-->  
    <link rel="preconnect" href="https://fonts.googleapis.com"> <!--Exemplo de uso de um <link>, conectando o documento com um recurso externo (nesse caso, google fonts). -->  
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <!--Função similar à linha de cima.-->  
    <link href="https://fonts.googleapis.com/css2?family=Lato:ital,wght@0,100;0,300;0,400;0,700;0,900;1,100;1,300;1,400;1,700;1,900&display=swap" rel="stylesheet"> <!--Aqui, os dados da fonte Lato são obtidos e pré-carregados, para que possa ser utilizado no CSS.-->  
    <title>Tags HTML</title> <!--Define o título da página na aba do navegador.-->  
    <link rel="stylesheet" href="tags_style.css"> <!--Conecta o arquivo HTML com o arquivo CSS.-->  
</head> <!--Fechamento da tag head.-->  
<body> <!--Abertura da tag body, que contém o conteúdo do site.-->  
    <h1>Olá!</h1> <!--Um título.-->  
</body> <!--Fechamento da tag body.-->  
</html> <!--Fechamento da tag html.-->

Olá!

Texto e Conteúdo

São tags responsáveis por organizar e formatar textos em uma página HTML. Com elas, é possível criar títulos, parágrafos, destaques em negrito ou itálico, quebras de linha, separadores visuais e blocos de conteúdo. São a base para exibir informações de forma clara e legível.

  • <h1></h1> até <h6></h6> - Títulos de diferentes níveis, diminuindo conforme a numeração.
  • <p>></p> - Define um parágrafo de texto.
  • <b>></b> ou <strong>></strong> - Negrito, colocado dentro de uma tag de texto (como p).
  • <em>></em> - Itálico, funciona da mesma forma que o negrito, dentro de uma tag de texto.
  • <div></div> - Container genérico em bloco, fundamental para estruturar áreas do site. Em uma analogia, seria como uma "caixa" que armazena conteúdo.
  • <br></br> - Quebra de linha. Não deve ser usado fora de tags de texto para definir espaços na página (para isso, use propriedades no CSS).
  • <span></span> - Container genérico em bloco que nem a div, mas em linha. Pode ser usado para estilizar partes específicas com CSS com uma class ou id.
  • <pre></pre> - Cria um texto pré-formatado que será mostrado de forma idêntica a como está escrito no arquivo .html., incluindo seus espaçamentos, quebra de linha etc.
  • <code></code> - Cria um texto com fonte monospace, servindo para representar um pedaço de código (os exemplos de código dessa página utilizam a tag code em conjunto com a tag pre!). Pode ser substituído por estilização em CSS.

Exemplo:

<h1>Título Principal</h1>
<p>Este é um <strong>texto em negrito</strong> e <em>itálico</em>.</p>
<p>Linha 1<br>Linha 2<br>Linha 3</p> <!-Parágrafo com 3 linhas.-->  
<p>Exemplo com <span style="color:red">destaque colorido</span> no meio do parágrafo.</p>
<hr> <!--Barra divisora.-->  
<div> <!--Abertura da tag div.-->  
    <p>Conteúdo dentro de uma div</p>
</div> <!--Fechamento da tag div.-->

Listas

As listas em HTML permitem organizar informações em sequência, seja com marcadores (não ordenadas), números (ordenadas) ou definições com descrições. Elas são úteis para apresentar passos, itens, conceitos ou agrupamentos de dados de forma estruturada e fácil de ler.

  • <ul></ul> - Cria uma lista não-numerada/ordenada. Alguns exemplos de uso seriam tópicos, bullet points etc.
  • <ol></ol> - Cria uma lista ordenada/numerada. Aqui, cada item recebe uma numeração que pode ser alterada e estilizada com CSS.
  • <li></li> - Cria um item dentro de uma lista. É usado tanto para a tag ul como para tag ol.
<ul>
    <li>Elemento</li>
    <li>Elemento</li>
    <li>Elemento</li>
</ul>
<ol>
    <li>Elemento</li>
    <li>Elemento</li>
    <li>Elemento</li>
</ol>

Mídias e Elementos

Essa categoria inclui os elementos usados para incorporar conteúdos interativos e visuais em uma página, como imagens, vídeos, áudios, links e iframes. São recursos que enriquecem o conteúdo, tornando-o mais dinâmico, visual e acessível ao usuário.

  • <img> - Exibe uma imagem, definida com src="". Pode ser uma imagem da internet, (ex: src="https://sitemuitolegal.com/imagens/gatoengraçado.jpg") ou da própria pasta do site, linkando ao caminho da imagem nos arquivos (ex: src="imagens/bolo.jpg").
  • <a></a> - Cria um link, definido com href="". Pode ser uma url (ex: href="https://wikipedia.org") ou uma página do próprio site (ex: href="index.html"). Além disso, o link pode ser aberto tanto na mesma guia do navegador (padrão, target="_self") ou em uma nova (target="_blank").

Semântica

As tags semânticas definem a estrutura lógica e o significado das seções de uma página. Elas não alteram as propriedades do conteúdo, mas com elas, é possível organizar o conteúdo em cabeçalho, rodapé, menu de navegação, áreas principais, seções e artigos. Isso melhora a leitura tanto para usuários quanto para mecanismos de busca.

  • <header></header> - Cabeçalho da página ou seção.
  • <footer></footer> - Rodapé, funciona de forma similar ao header.
  • <main></main> - Conteúdo principal da página, só pode haver um por página.
  • <section></section> - Uma seção da página.
  • <article></article> - Uma seção "independente" da página que "faz sentido sozinha", como posts de blog.
  • <nav></nav> - Uma área de navegação, como menus, topbars etc.

Exemplo:

<header>
    <h1>Meu Site</h1> <!--Um Título-->
<header>
<nav>
    <ul>
        <li>Início</li>
        <li>Sobre</li>
    </ul>
</nav>
<main> <!--Abertura da tag main-->
    <section> <!--Abertura da tag de seção-->
            <h2>Notícias</h2> <!--Um subtítulo-->

        <article>
            <h3>Evento importante</h3>
            <p>Hoje ocorreu um evento significativo.</p>
        </article>
    </section> <!--Fechamento da tag de seção-->
</main> <!--Fechamento da tag main-->

<footer> <!--Abertura da tag de rodapé-->
    <p>© 2025 Meu Site</p> <!--Conteúdo do rodapé-->
</footer> <!--Fechamento da tag de rodapé-->

Botões e Formulários

São tags usadas para criar áreas de interação onde o usuário pode inserir informações ou executar ações. Essas tags permitem desde a digitação de um texto simples até a seleção de opções, envio de dados e execução de comandos na página. Combinando elementos como campos de entrada, listas suspensas, áreas de texto e botões, é possível desenvolver sistemas de login, pesquisas, cadastros e qualquer recurso que dependa de dados fornecidos pelo usuário.

  • <button></button> - Cria um botão clicável, que pode levar à uma página caso uma tag a seja colocadada dentro dele (ex: <button><a href="...">Clique</a></button>).
  • <input>- Define um campo onde o usuário pode colocar alguma informação. Pode ser formatado com type="", com uma lista vasta de possibilidades. Desde checkbox, image, color, date, email etc. Clique aqui para ver uma lista completa.
  • <form></form> - Usado para uma ação do usuário, aninha outras tags dentro de si como o próprio input, textarea, label etc. Leva a informação à uma página, definida com action=""
  • <textarea></textarea> - Usado para uma seção de texto escrito pelo usuário, que pode ser enviada.

Exemplo:

<form> <!--Abre o formulário.--> 
    <textarea rows="5" cols="30"> <!--Cria uma seção de texto editável pelo usuário, com 5 linhas e 30 colunas, de nome teste e id teste_1.--> 
    <button><a href="sessentaeseis.github.io>Clique!></a></button> <!--Botão clicável.--> 
    <input type="submit" value="submit"> <!--Botão de enviar, ao final.--> 
</form> <!--Fechamento da tag form.-->
